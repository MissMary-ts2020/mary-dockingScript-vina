## 用法
> 本工具宜于业已位点预测之场合
- zinc数据之库宜备毕，并副本于工作目录之下为zinc.bak之目录
- 位点预测之数应如实列于config.conf文件之中
- 受体蛋白应备妥，并依其名号书于config文件之中
- 运行一次flash脚本

> 参数说明详见help.txt或以-h参数运行之，以查看
